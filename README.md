# PhoneNumberChecker

App to validate phone numbers

# Tech Stack
- ReactJS
- Dotnet Core
- TailwindCSS
- Material UI
- Vite

# Frontend

## Install the dependencies
```bash
yarn
```

### Start the app in development mode (hot-code reloading, error reporting, etc.)
```bash
yarn dev
```

### Build the app for production
```bash
yarn build
```

### Run app in production
```bash
yarn serve
```

# Backend

### Start the app
```bash
dotnet run
```

### Build the app
```bash
dotnet build
```

### Publish the app for production
```bash
dotnet publish
```
